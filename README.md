# DekhoDeal

Telegram-first affiliate deals platform for `dekhodeal.website`.

## Live Flow
- Real-time deals are posted on Telegram channel:
  - `https://t.me/DekhoDeal`
- Website is used for trust pages and channel discovery.

## Pages
- `index.html` - Landing page + Telegram CTA
- `products.html` - Notice page (deals moved to Telegram)
- `legal.html` - Affiliate disclosure and legal notes
- `contact.html` - Contact and business queries

## Assets
- `logo.svg`
- `favicon.svg`
- `styles.css`
- `script.js`
- `sitemap.xml`
- `robots.txt`
- `CNAME`

## Deploy (GitHub Pages)
1. Push all files to `main` branch.
2. GitHub repo -> Settings -> Pages.
3. Source: `Deploy from a branch`, Branch: `main`, Folder: `/ (root)`.
4. Custom domain: `dekhodeal.website`.
5. Enable HTTPS after certificate is issued.

## Update Routine
- Post deals directly in Telegram channel.
- Keep legal/contact details on website updated.
