# DekhoDeal

Simple static affiliate website for GitHub Pages.

## Deploy on GitHub Pages
1. Create a new GitHub repo (public).
2. Upload these files to the repo root: `index.html`, `styles.css`, `script.js`, `CNAME`.
3. Go to Settings > Pages.
4. Source: `Deploy from a branch`.
5. Branch: `main` and folder `/root`.
6. Save. It may take a few minutes to publish.

## Custom Domain
This repo includes a `CNAME` file for `dekhodeal.website`.
Set DNS at your domain provider using the official GitHub Pages documentation
for the current IPs and required records.

## Edit content
Update copy and replace `#` links with your affiliate URLs.
